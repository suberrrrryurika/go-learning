### 1. Go 语言简介（30 分钟）

#### 历史背景
```
Go 语言由 Google 于 2007 年开始设计，2009 年发布
设计者：Robert Griesemer, Rob Pike, Ken Thompson
目标：结合 C++ 的性能和 Python 的简洁
```

#### 主要特点
- **简洁快速**: 编译速度快，语法简洁
- **并发强大**: goroutine 和 channel
- **类型安全**: 静态类型，类型推断
- **垃圾回收**: 自动内存管理
- **跨平台**: Windows/Linux/Mac 原生支持

#### 应用领域
```
✓ Web 后端开发
✓ 微服务架构
✓ 云原生应用
✓ 命令行工具
✓ 分布式系统
```

### 2. 环境安装（45 分钟）

#### Linux (Ubuntu/Debian)
```bash
# 下载 Go 1.21
wget https://go.dev/dl/go1.21.5.linux-amd64.tar.gz

# 解压到 /usr/local
sudo tar -C /usr/local -xzf go1.21.5.linux-amd64.tar.gz

# 配置环境变量
echo 'export PATH=$PATH:/usr/local/go/bin' >> ~/.bashrc
echo 'export GOPATH=$HOME/go' >> ~/.bashrc
echo 'export PATH=$PATH:$GOPATH/bin' >> ~/.bashrc
source ~/.bashrc

# 验证安装
go version
```

#### macOS
```bash
# 使用 Homebrew
brew install go@1.21

# 或下载安装包
# https://go.dev/dl/go1.21.5.darwin-amd64.pkg

# 验证
go version
```

#### Windows
```
1. 下载安装包：https://go.dev/dl/go1.21.5.windows-amd64.exe
2. 运行安装程序
3. 默认安装到 C:\Go
4. 系统自动配置环境变量
5. 打开 CMD 验证：go version
```

#### 示例 2: 变量与数据类型
```go
// 文件：variables.go
package main

import "fmt"

func main() {
    // 变量声明
    var name string = "Go 语言"
    var version float64 = 1.21
    var year int = 2026
    var isAwesome bool = true
    
    // 类型推断
    var inferred = "自动推断类型"
    short := "简短声明"  // := 是常用的声明方式
    
    // 常量
    const Pi = 3.14159
    const MaxUsers = 1000
    
    // 输出
    fmt.Println("=== Go 语言信息 ===")
    fmt.Println("名称:", name)
    fmt.Println("版本:", version)
    fmt.Println("年份:", year)
    fmt.Println("很棒吗？", isAwesome)
    fmt.Println("推断类型:", inferred)
    fmt.Println("简短声明:", short)
    fmt.Printf("圆周率：%.2f\n", Pi)
    
    // 格式化输出
    fmt.Printf("名称类型：%T\n", name)
    fmt.Printf("版本类型：%T\n", version)
}
```

**运行结果**:
```
=== Go 语言信息 ===
名称：Go 语言
版本：1.21
年份：2026
很棒吗？ true
推断类型：自动推断类型
简短声明：简短声明
圆周率：3.14
名称类型：string
版本类型：float64
```

### 4. Go 项目结构（45 分钟）

#### 标准项目结构
```
myproject/
├── go.mod              # 模块定义
├── go.sum              # 依赖校验
├── main.go             # 主程序入口
├── cmd/                # 命令行程序
│   └── server/
│       └── main.go
├── internal/           # 私有包
│   └── config/
│       └── config.go
├── pkg/                # 公共包
│   └── utils/
│       └── helper.go
├── api/                # API 定义
├── web/                # Web 资源
├── test/               # 测试文件
└── docs/               # 文档
```

#### 创建项目实战
```bash
# 创建项目目录
mkdir -p ~/go/projects/hello-go
cd ~/go/projects/hello-go

# 初始化模块
go mod init hello-go

# 创建主文件
cat > main.go << 'EOF'
package main

import "fmt"

func main() {
    fmt.Println("我的第一个 Go 项目！")
}
EOF

# 运行
go run main.go

# 编译
go build -o hello main.go
./hello
```

### 练习 2: 简单计算器
```go
// 任务：实现一个简单的四则运算计算器

package main

import (
    "fmt"
    "bufio"
    "os"
    "strings"
    "strconv"
)

func main() {
    reader := bufio.NewReader(os.Stdin)
    
    fmt.Print("请输入第一个数字：")
    num1Str, _ := reader.ReadString('\n')
    num1, _ := strconv.ParseFloat(strings.TrimSpace(num1Str), 64)
    
    fmt.Print("请输入第二个数字：")
    num2Str, _ := reader.ReadString('\n')
    num2, _ := strconv.ParseFloat(strings.TrimSpace(num2Str), 64)
    
    fmt.Println("\n选择运算：")
    fmt.Println("1. 加法 (+)")
    fmt.Println("2. 减法 (-)")
    fmt.Println("3. 乘法 (*)")
    fmt.Println("4. 除法 (/)")
    
    fmt.Print("请输入选项 (1-4): ")
    option, _ := reader.ReadString('\n')
    option = strings.TrimSpace(option)
    
    var result float64
    var op string
    
    switch option {
    case "1":
        result = num1 + num2
        op = "+"
    case "2":
        result = num1 - num2
        op = "-"
    case "3":
        result = num1 * num2
        op = "*"
    case "4":
        if num2 == 0 {
            fmt.Println("错误：除数不能为 0！")
            return
        }
        result = num1 / num2
        op = "/"
    default:
        fmt.Println("无效选项！")
        return
    }
    
    fmt.Printf("\n结果：%.2f %s %.2f = %.2f\n", num1, op, num2, result)
}
```

### 练习 3: 温度转换器
```go
// 任务：摄氏度和华氏度互相转换
// 公式：F = C × 9/5 + 32
//      C = (F - 32) × 5/9

package main

import "fmt"

func celsiusToFahrenheit(c float64) float64 {
    return c * 9.0 / 5.0 + 32
}

func fahrenheitToCelsius(f float64) float64 {
    return (f - 32) * 5.0 / 9.0
}

func main() {
    fmt.Println("=== 温度转换器 ===\n")
    
    // 摄氏度转华氏度
    fmt.Println("摄氏度 → 华氏度:")
    for c := 0; c <= 100; c += 10 {
        f := celsiusToFahrenheit(float64(c))
        fmt.Printf("%3d°C = %6.2f°F\n", c, f)
    }
    
    fmt.Println("\n华氏度 → 摄氏度:")
    for f := 32; f <= 212; f += 18 {
        c := fahrenheitToCelsius(float64(f))
        fmt.Printf("%3d°F = %6.2f°C\n", f, c)
    }
}
```
