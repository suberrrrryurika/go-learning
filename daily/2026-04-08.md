## 💻 代码示例

### 示例 1: 基础示例

```go
package main

import "fmt"

func main() {
    // 变量声明
    var name string = "Go 语言"
    version := 1.21
    
    // 输出
    fmt.Printf("欢迎学习 %s v%.2f\n", name, version)
    
    // 数据类型演示
    var (
        age     int     = 25
        height  float64 = 175.5
        isStudent bool  = true
    )
    
    fmt.Printf("年龄：%d, 身高：%.1f, 学生：%v\n", age, height, isStudent)
}
```

**运行结果**:
```
欢迎学习 Go 语言 v1.21
年龄：25, 身高：175.5, 学生：true
```

### 示例 2: 进阶示例

```go
package main

import "fmt"

// 数组与切片
func arraySliceDemo() {
    // 数组
    var arr [5]int = [5]int{1, 2, 3, 4, 5}
    fmt.Printf("数组：%v\n", arr)
    
    // 切片
    slice := []int{10, 20, 30, 40, 50}
    fmt.Printf("切片：%v\n", slice)
    
    // 切片操作
    fmt.Printf("切片 [1:3]: %v\n", slice[1:3])
    fmt.Printf("切片 [:3]: %v\n", slice[:3])
    fmt.Printf("切片 [2:]: %v\n", slice[2:])
    
    // append
    slice = append(slice, 60)
    fmt.Printf("append 后：%v\n", slice)
}

func main() {
    arraySliceDemo()
}
```
