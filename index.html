第 1 天：Go 语言简介与环境搭建

学习内容

  • 了解 Go 语言的历史和特点
  • 安装 Go 开发环境（Linux/Mac/Windows）
  • 配置 GOPATH 和 GOMOD
  • 编写第一个 Hello World 程序
  • 学习使用 go run 和 go build 命令

实践任务

  • 安装 Go 1.21+ 版本
  • 配置环境变量
  • 创建 hello.go 文件
  • 运行并编译程序
  • 理解 go.mod 文件
